About Pax Consola

Pax Consola is being presented at the Vintage Computer Festival East in 2021.

Developed by Tim Ryan.

How to use Pax Consola

paxconsola.com consists of a single-pane coding input and an emulator sidebar. You are able to write a program in Pax Forth (a simple Forth dialect) that can run on many different vintage computers and consoles. The underlying graphics system is implemented for many different machines, so you only need to write your game or script once.

After modifying the code, use the sidebar on the right to select which console you'd like to run the game on.

Credits

Pax Consola is inspired by the BASIC systems and Game Maker programs I spent hours on as a kid. As well as my dad, who helped me appreciate how fun it is to build new things.

Pax Forth was inspired in part by Lars Brinkhoff's nybbleForth stack machine.

I credit learning Forth from the tutorial for Easy Forth by Nick Morgan, which the Snake game was derived from.

Pax Consola uses the Emularity emulator system from The Internet Archive as well as WasmBoy.
