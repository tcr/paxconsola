# Pax Consola

An implementation of the forth programming language intended for cross-compilation.

Some things you can run:

### Compile Snake

```sh
cargo run -- run snake/gb/snake.pax
```

### Run snake on Gameboy

```
cd snake/gb
make run
```

### Run the paxconsola IDE

```
cd paxconsola-ide
./deploy.sh
```
