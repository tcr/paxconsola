
$C004 constant last-key
$C006 constant temp
$9800 constant graphics

37 constant left
38 constant up
39 constant right
40 constant down

: drop    if then ;
: over   >r temp ! temp @ r> temp @ ;

: move-right 18 graphics 3 + ! ;
: move-left 21 graphics ! ;
: move-up 19 graphics 2 + ! ;
: move-down 20 graphics 1 + ! ;

last-key @ 
  left over = if move-left else
  up over    = if move-up else
  right over = if move-right else
  down over  = if move-down
  then then then then
